# Claude Code Subagents

This folder is intentionally scaffolded with guidance only.

Claude Code supports specialized subagents under:

- `.claude/agents/`

## What Subagents Are For

Use subagents to separate responsibilities, for example:

- architecture and planning
- implementation for specific domains
- test strategy and quality checks
- security and compliance review

## How To Start

1. Review the official Claude Code documentation.
2. Define one subagent with a clear scope and constraints.
3. Iterate based on team usage and outcomes.
