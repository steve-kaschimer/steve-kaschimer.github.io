# Kiro Agents

This folder is intentionally scaffolded with guidance only.

Kiro supports custom agents and subagent-style workflows.

## What To Use This For

Use custom agents to specialize behavior, such as:

- feature planning
- implementation execution
- code review and risk checks
- migration and refactor support

## How To Start

1. Review current Kiro docs/changelog for agent features.
2. Add one focused custom agent and validate its behavior on a real task.
3. Keep durable project rules in `AGENTS.md`, `docs/`, and `.specify/`.
